# Project Overiew
this project performs data analysis on a retail sales database using sql. the main goal of this project is to get the insights like store performance, product popularity, customer behaviour, brand insights and low stack alert. 
I have used the concept of sql joins, aggregation, grouping and sorting to got all these insights. This project is one of my earlier ones. 

## DATABASE INFORMATION
this dataset contians the information about stores, customers, orders products, brands etc. 
| Table       | Description                       |
| ----------- | --------------------------------- |
| stores      | Store information                 |
| staffs      | Staff members working at stores   |
| customers   | Customer details                  |
| orders      | Customer orders                   |
| order_items | Products included in each order   |
| products    | Product information               |
| brands      | Product brand details             |
| categories  | Product categories                |
| stocks      | Inventory available in each store |
